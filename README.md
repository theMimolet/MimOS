# MimOS

MimOS is a home-made version of Bazzite that's made to fit my needs. (Dev + gaming on a recent Nvidia card)

## Why ?

I used to be on Bazzite DX and it was awesome.

However, it's based on the Bazzite-Deck images. I don't personally work on a Steam Deck, and so I don't need all the deck related compatibility and software additions. Moreover, the decks builds are on hold for now because they are overhauling the deck specific features - and knowing there's a vulnerability with the current on the image, I don't want to be stuck with it.

So I decided to make my own spin, based on Bazzite stable, that removes what I don't need and adds what I do need.
